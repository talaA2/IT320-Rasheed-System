# IT320-Rasheed-System

## Overview
Rasheed is a web-based reporting system designed to help residents report water and electricity issues in their communities. The system allows users to submit reports, track their status, receive notifications, and view rewards. It also provides an admin interface for reviewing reports and updating their status.

## Project Objectives
- Provide a unified platform for reporting utility issues
- Improve transparency through report tracking
- Encourage community participation using a reward system
- Support administrators in managing submitted reports

## Main Features

### Resident Features
- Register an account
- Log in and log out
- Add a report
- View and track submitted reports
- Receive notifications
- View reward points and badges

### Admin Features
- View all submitted reports
- Update report status
- Trigger reward assignment when a report is marked as completed
